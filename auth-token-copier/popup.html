Auth Token Copier

This extension copies the authentication token from the source URL's network headers and stores it in the destination URL's local storage.

Modes explanation:

  • OFF: Extension does nothing automatically. You can still manually paste in a token and write it to the destination's local storage by clicking Write.
  • MANUAL: Extension monitors network requests and keeps the token field updated with the latest token. You must manually click Write to write it to local storage.
  • AUTO: Same as above, except extension also writes tokens to local storage automatically as they are obtained.
